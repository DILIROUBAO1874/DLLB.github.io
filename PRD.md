## 2. 设计理念

### 2.1 动态品牌符号 (The Dynamic Monogram)

**图形设计：**
- 由细线构成的正方形，内部由一条对角线切分
- 正方形代表场域（Site），切分线代表介入（Intervention）
- 不代表具体的房子，而代表处理空间的方法论

**交互细节：**
- 用户进入网页时，符号有极简的生长动画（线条从一点向四周蔓延）
- 动画完成后固定在左上角作为全局导航按钮
- 默认 30% 透明度，悬停时恢复至 100%

### 2.2 硫酸纸图层叙事 (The Layering Narrative)

针对全专业（规划/建筑/景观/室内）的展示逻辑，采用**"尺度切换"**的叙事方式：

**视觉分层：**
- 每一屏代表一个项目，全屏展示
- 背景色：带颗粒感的暖白（#F9F7F2）

**三层叠加：**
- **第一层（底层）：** 淡淡的网格线或总平面图
- **第二层（中间层）：** 技术指标（尺度、地点等）像打印机一样缓慢打出
- **第三层（最顶层）：** 实景/效果图以 0.8s 淡入效果覆盖

**专业差异：**
- 规划项目：展示宏观的纹理与逻辑
- 室内项目：展示微观的材质与光影

## 3. 功能需求

### 3.1 页面结构（单页滚动）

```
Logo 生长动画（入场）
    ↓
Hero 首屏（全屏）
    ↓
项目 1（全屏，三层叙事）
    ↓
项目 2（全屏，三层叙事）
    ↓
项目 3 ... 项目 N
    ↓
关于我们
    ↓
联系方式
    ↓
页脚
```

### 3.2 动态品牌符号 (Logo)

| 需求项 | 说明 |
|--------|------|
| 形态 | 细线正方形 + 对角线切分 |
| 入场动画 | 线条从中心点向四周蔓延生长（1.5s） |
| 固定位置 | 左上角，position: fixed |
| 透明度 | 默认 30%，悬停 100% |
| 点击行为 | 回到页面顶部 |

### 3.3 导航栏

| 需求项 | 说明 |
|--------|------|
| 位置 | 固定在页面顶部右侧 |
| 内容 | 导航链接（首页 / 作品 / 关于 / 联系） |
| 行为 | 点击导航链接平滑滚动到对应区域 |
| 滚动效果 | 页面向下滚动后，背景变为毛玻璃半透明 |

### 3.4 Hero 首屏

| 需求项 | 说明 |
|--------|------|
| 布局 | 全屏展示，占据 100vh |
| 背景 | 带颗粒感的暖白底色 + 淡网格线 |
| 内容 | 工作室名称「W² Studio」+ 一句中英混排理念标语 |
| 动画 | 文字缓慢淡入 |
| 滚动提示 | 底部向下箭头，呼吸动画 |

### 3.5 项目展示（核心）

每个项目占据一整屏，采用三层叙事：

| 层级 | 内容 | 动画 |
|------|------|------|
| 第一层 | 淡网格线背景 | 始终存在 |
| 第二层 | 项目名称 + 地点 + 规模 + 年份 | 打字机效果，逐字显示 |
| 第三层 | 效果图全屏覆盖 | 0.8s 淡入，覆盖前两层 |

**交互：**
- 滚动到该项目区域时自动触发第二层打字机动画
- 打字完成后触发第三层效果图淡入
- 点击效果图可打开详情弹窗

## 4. 设计规范

### 4.1 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景主色 | `#F9F7F2` | 带颗粒感的暖白 |
| 文字主色 | `#1A1A1A` | 近黑 |
| 文字次要 | `#666666` | 中灰 |
| 分割线/网格线 | `#E0E0E0` | 浅灰，低透明度 |
| Logo 线条 | `#1A1A1A` | 30% 透明度 |
| 遮罩层 | `rgba(0,0,0,0.7)` | 弹窗背景 |

### 4.2 颗粒感纹理

- 使用 CSS `background-image` 叠加细微噪点纹理
- 透明度极低（~3%），营造硫酸纸质感
- 纯 CSS 实现，不依赖外部图片

### 4.3 字体

| 用途 | 字体 | 来源 |
|------|------|------|
| 英文 | Inter | Google Fonts |
| 中文 | Noto Sans SC（思源黑体） | Google Fonts |
| 标题 | font-weight: 300（细体） | — |
| 正文 | font-weight: 400（常规） | — |
| 打字机文字 | 等宽字体（monospace）备选 | — |

### 4.4 Logo 设计 — 动态品牌符号

- **形态：** 细线（1px）构成的正方形 + 对角线切分
- **线条粗细：** 1px，精致感
- **入场动画：** 线条从中心点向四周蔓延生长（clip-path 或 stroke-dasharray）
- **动画时长：** 1.5s，ease-out
- **固定位置：** 左上角 (20px, 20px)，position: fixed
- **默认透明度：** 30%，hover 时 100%
- **尺寸：** 40px × 40px
- **配色：** 单色（`#1A1A1A`）

### 4.5 响应式断点

| 断点 | 设备 | 项目布局 |
|------|------|----------|
| ≥ 1200px | 桌面端 | 全屏滚动，打字机+效果图叠加 |
| 768px ~ 1199px | 平板端 | 全屏滚动，适配字号 |
| < 768px | 移动端 | 全屏滚动，竖向排版 |

## 5. 动画规范

| 元素 | 动画效果 | 触发条件 | 时长 |
|------|----------|----------|------|
| Logo 符号 | 线条从中心向四周蔓延生长 | 页面加载 | 1.5s |
| Logo 透明度 | 30% → 100% | 鼠标悬停 | 0.3s |
| Hero 文字 | 缓慢淡入 | 页面加载 | 1s |
| 向下箭头 | 上下浮动呼吸动画 | 页面加载，循环 | 2s |
| 导航栏 | 背景毛玻璃 + 阴影 | 向下滚动 > 50px | 0.3s |
| 项目第一层 | 淡网格线始终存在 | — | — |
| 项目第二层 | 打字机逐字显示技术指标 | Intersection Observer 检测进入视口 | 1.5s |
| 项目第三层 | 效果图全屏淡入覆盖 | 打字机完成后 | 0.8s |
| 关于/联系区域 | 从左/右滑入淡出 | Intersection Observer | 0.8s |
